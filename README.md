## ⚡ System Overview
Mainstream IDEs have become bloated with forced telemetry, cloud subscriptions, and heavy extensions. **HVX Editor Pro** strips away the noise. It is a lightning-fast, 100% offline development environment designed for developers who care about aesthetics, performance, and the pure hacking experience.

## 🚀 Key Features

* **Hardware Telemetry HUD:** Reads your motherboard directly. Live RAM allocation and CPU thread tracking piped straight into your right-side HUD without requiring any external plugins.
* **Zero-Bloat Architecture:** Loads instantly. Only parses the languages you actually use (C, Python, JS, HTML/CSS).
* **Integrated Execution Engine:** Hardcoded macros allow you to compile and run `C` and `Python` scripts with a single press of `F5`, piping the output directly to the built-in system terminal.
* **CONFIG_OS Customization:** Take full control of the UI. Adjust the Neon Hex color, Glass Depth opacity, and Matrix Rain speed. Your vibe is saved directly to your local config.
* **Smart File System:** Features inline renaming, secure deletion protocols, and ghost-file crash protection.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Core Engine:** Electron & Node.js
* **Code Matrix:** Monaco Editor (VS Code's core parser, stripped of the bloat)

## 📥 Installation
You do not need to build this from source to use it. 
1. Head over to the [Releases](../../releases) tab.
2. Download the latest `HVX Editor Pro Setup.exe`.
3. Run the installer and enter the Matrix.

*(Currently supported on Windows 10 & 11 64-bit).*
